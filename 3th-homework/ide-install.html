既然选择后台开发，强力建议你在 Linux 环境下安装 go 语言开发环境。这里仅是 centos 7 的部分命令，Ubuntu 等的支持请通过网上资源或同学的博客。

2、安装 atom / vs-code

请参考官方网站，完成对应平台的安装。Atom 在 contos 7 的安装：

$ sudo yum install atom.x86_64.rpm

3、安装 golang

Golang 官方网站 提供了不同平台的安装。golang 中国项目组 提供了近可能好的中文服务。如果你有兴趣，发现问题可联系它们，使得中文服务变得更加完善。

3.1 安装

中文安装指南位置：https://golang.org/doc/install

3.2 设置环境变量

go 对编译、包管理、测试、部署、运行提供全程支持，了解环境配置非常重要！

需要配置的环境变量，对于 centos 在 /etc/profile 或 ~/.profile 文件中添加:

#export GOROOT=/usr/local/go
export GOROOT=/opt/go
export GOBIN=$GOROOT/bin
export PATH=$PATH:$GOBIN
#export GOPATH=~/go-project/

3.3 检查安装

使用最简单的 GO 命令

$ go version
$ go env

要点:

$GOBIN 目录，存放了 go 语言的实用程序
$GOPATH 目录，通常是你 GIT 本地工作区的目录，或 应用的工作目录。 你导入的应用程序库都在这个目录下。默认目录 $HOME/go

3.4 hello world！

使用 vi 或 notepad++ 这类工具创建文件 hello.go (注意中文使用 utf-8 编码)：

package main

import "fmt"

func main() {
    fmt.Printf("hello, world\n")
}

运行!

$ go run hello.go
hello, world

4.1 修改 $GOBIN 目录权限

sudo chmod 777 $GOBIN

因为插件需要在 $GOBIN 中安装扩展实用程序，而编辑器一般运行于普通用户态

4.3 解决问题

GO 工具包安装方法

参考 https://studygolang.com/articles/10263

找GOPATH的位置

[~]$ go env
...
GOPATH="..."
...

获得帮助

[~]$ go help get

下载源代码到本地

# 创建文件夹
mkdir $GOPATH/src/golang.org/x/
# 进入文件夹
cd $GOPATH/src/golang.org/x/
# 下载源码
git clone https://github.com/golang/tools.git

安装工具包

go install golang.org/x/tools/cmd/goimports
go install golang.org/x/tools/cmd/gorename
go install golang.org/x/tools/cmd/guru
# 强力建议先下载，后安装。避免 get 的等待
...

安装完成后，就 $GOROOT/bin 下看到这些执行文件
